# Roblox Sound ID Detector

Web tool untuk mendeteksi dan mengekstrak seluruh Sound Asset ID dari inventory akun Roblox.

## Cara Deploy di Railway
1. Push repository ini ke GitHub.
2. Buat project baru di Railway dan connect ke repo ini.
3. Pergi ke tab **Variables** di service Railway lu.
4. Tambahkan variable baru:
   - Name: `ROBLOX_COOKIE`
   - Value: `[Tempel cookie .ROBLOSECURITY akun alt lu di sini]`
5. Railway akan otomatis restart dan web lu langsung online!
